CSS tiene una estructura bien definida para agregar estilos a nuestros elementos html. Tenemos los elementos siguientes: selector, propiedad y valor

P{color: red;}

En este ejemplo: 10px es el offset X. 5px es el offset Y. 5px es el blur radius. 0px es el spread radius. rgba(0, 0, 0, 0.75) es el color de la sombra.
